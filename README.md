##Plushie's Textures
A Balatro modpack of texture mods I've made.

##Currently includes:

#Oops! All Jimbos
Makes every joker look like jimbo. It's cursed by design.

#Plushie's Legendary Backgrounds
Changes the backgrounds of the 5 basegame legendary jokers to be unique from each other, and based on the base 4 suits and adding in stars for Canio.
